Special terms and conditions for category Tablet Based Digital Monitoring and
Attendance System 

1.Pre-loaded cloud based management software should be capable of contextually managing
and monitoring the devices, pushing applications and content, pushing group based
messages, managing through a remote web based control centre and provide web based
MIS dashboard. The management software will allow to manage the contexts like date &
time of use, network, geo-locations etc. in both online and offline mode. The devices should
be provisioned for providing remote and automatic updates and upgrades from a centralized
web based interface. The end users should not be able to uninstall this software or any
application. The software will prevent the tablet from use by un-authorized user in case it is
stolen and help in tracing the location. The Logo & message of the user Department for the
users will be included in the Boot software so that it will appear on the Tablet any time it is
switched on for use.

2. If Seller is uploading product with agreement to print use’s specific logo, they will print the
same as per design, content and style provided at by the buyer at time placement of
purchase order without any extra cost.

3.Tablet Based Biometric Attendance System shouldbe unrooted and Safety Net passed.
